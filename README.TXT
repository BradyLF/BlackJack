PROJECT TITLE: Black Jack at the Casino BlueJ

PURPOSE OF PROJECT: to create a functional program that can play like a regular blackjack game,

VERSION or DATE: version 1.0 code finalized on January 8 of 2015

AUTHORS: Brady Africk, Steve Balmer, Binky

USER INSTRUCTIONS: 

open the package.bluej file. 
right click game, 
click new game. 
go down to the red square, 
right click. 
click play.

You'll need BlueJ as well as Java SDK 1.6 or later
